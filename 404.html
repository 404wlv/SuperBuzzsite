Internal system overview for the 404 development team

Overview

SuperBuzz is a university web platform built for the University of Wolverhampton. It integrates events, maps, chat, transport, and authentication into a single system.

Architecture

Frontend-heavy modular JavaScript system powered by Supabase.

  • home.js → main dashboard logic
  • map.js → 3D campus map system
  • logon.js → authentication flow
  • Supabase → database + auth

Events System

Handles event creation, retrieval, and UI rendering.

loadEvents()
renderEvents()
openEventModal(event)

Data stored in Supabase 'events' table.

Chat System

Keyword-based FAQ chatbot system.

sendMessage()
getBotReply(message)

Matches user input against FAQ keywords.

Map System

3D campus visualization using MapLibre GL JS.

fetchbuildings()
parse_osm_data()
getCentroid()
openDrawer()

Uses Overpass API + GeoJSON conversion.
