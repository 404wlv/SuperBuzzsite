SuperBuzz Developer Documentation

Internal system overview for Team 404

Overview

SuperBuzz is a modular university platform built for the University of Wolverhampton. It combines events, navigation, chat, authentication, profile management, and gamification systems into one unified web application.

Architecture

Frontend modular JavaScript system powered by Supabase backend-as-a-service.

  • home.js → main dashboard (events, chat, UI)
  • map.js → 3D campus map system
  • logon.js → authentication system
  • profile.js → user profile management
  • dailyCheckin.js → gamified streak system
  • Supabase → database + authentication
