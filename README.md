# SuperBuzzsite

Website version of SuperBuzz Mobile App

This website is an interactive website for University of Wolverhampton students to make their lives easier.

Main features:
- All the events going around campus(City Campus + Wallsal Campus)
- Add an event if you wish to host one -> will be approved by Admin
- Chat with an AI/FAQs bot for help or any questions you have
- Find out LIVE bus timings, WLVGYM timetable, Harrison and Walsall Library timetable
- Campus Map with details on each building.
